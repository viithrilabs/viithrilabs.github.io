POBOT

An intelligent automation tool built to eliminate repetitive workflows at ViiThri Labs. What started as a script to save 20 minutes per day became the core of how the studio operates.

The problem

Every day at ViiThri Labs involved the same tasks — checking task statuses across tools, sending update messages, managing file organisation, triggering builds, and tracking time. Each task took minutes. Together, they consumed hours per week.

No existing tool handled all of it without significant manual configuration. So we built POBOT — named for the repetitive bot-like nature of the work it replaced.

What it does
Task Scheduling
Time-based and event-driven task execution with cron-style configuration
File Management
Automated organisation, archiving, and sync across local and cloud storage
API Integrations
Connects to GitHub, Notion, email, and custom endpoints
Notification System
Smart alerts via email and messaging platforms with configurable rules
Report Generation
Automated daily and weekly summaries from multiple data sources
CLI Interface
Lightweight command-line interface for triggering tasks on demand
Tech stack
Python 3.11 APScheduler Click (CLI) HTTPX SQLite Jinja2 (templating) GitHub Actions YAML config
