Studio Dashboard

Internal tooling built specifically for how ViiThri Labs operates — project pipelines, client management, invoicing, and timelines, all in one place. We eat our own cooking.

Why we built it

Running a studio means tracking multiple projects simultaneously — each with its own deadlines, client contacts, invoice schedules, and deliverables. Generic project management tools are either too broad or require significant configuration to match how a small studio actually works.

Studio Dashboard is purpose-built for ViiThri Labs. It knows our project types, our billing model, and our workflow. It is also the clearest demonstration of what we offer under the Digital Tools service — we build for ourselves with the same standards we apply to client work.

Core features
Project Pipeline
Kanban-style view of all active projects with status, deadline, and health indicators
Client Records
Centralised contact database with project history and communication log
Invoice Management
Create, send, and track invoices with payment status and automatic reminders
Time Tracking
Per-project time logging linked to billing and profitability reports
Deliverable Tracker
Checklist-based milestone tracking with client-facing progress view
Revenue Dashboard
Monthly revenue, pipeline value, and projected earnings at a glance
Tech stack
Python / FastAPI React PostgreSQL Redis Docker Razorpay (billing) SMTP (email)
